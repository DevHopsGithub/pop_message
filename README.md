Simple popup message for Flutter.

## Installing

1. Add dependencies to `pubspec.yaml`

    ```yaml
    dependencies:
        cronos:
            git:
                url: https://github.com/DevHopsGithub/pop_message
    ```

2. Run pub get.

   ```shell
   flutter pub get
   ```

3. Import package.

    ```dart
    import 'package:pop_message/pop_message.dart';
    ```
